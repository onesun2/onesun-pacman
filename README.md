# Neon Pacman

A classic Pacman-style game built with HTML5 Canvas and JavaScript, featuring a modern neon aesthetic and 3 progressive stages.

## Features

- **3 Unique Stages**: Increasing difficulty and complex layouts.
- **Neon Visuals**: Glowing retro aesthetic.
- **Classic Gameplay**: Eat dots, avoid ghosts, and use power pellets to turn the tables.
- **Responsive Controls**: Smooth keyboard controls.

## How to Play

1.  Open the game (or visit the deployment link).
2.  Press **SPACE** to start.
3.  Use **Arrow Keys** to move.
4.  Clear all dots to advance.

## Development

This project uses vanilla HTML, CSS, and JavaScript. No build steps required.

### Files
- `index.html`: Main entry point.
- `style.css`: Visual styling.
- `script.js`: Game logic.
